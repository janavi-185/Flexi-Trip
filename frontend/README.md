# Tax Loss Harvesting Tool

A React-based dashboard for analyzing crypto holdings and optimizing tax strategies through loss harvesting. Displays real-time capital gains/losses calculations and allows users to simulate tax-saving scenarios by selecting holdings.

**Tech Stack**: Nextjs, TypeScript, Tailwind CSS, Shadcn/UI, Context API, 

**Setup**: `npm install` → `npm run dev` | **Deploy**: Vercel/Netlify

Track your gains, harvest your losses, save on taxes.
